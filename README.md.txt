Calculator
Este proyecto tiene como objetivo crear una calculadora funiciones
básicas como:

1.Addition
2.Subtration
3.Division
4.Multiplication
5.Module
6.Sin
7.Cos
8.Tan
9.Logarithm Base10
10.Logarithm base n
11.Square Root
12.n Root
13.Potency.
14.Base10 of a number
15.Factorial

Y conversiones como:

1.Conversion degrees to radians
2.Conversion radians to degrees
3.Conversion binary to hexadecimal
4.Conversion de hexadecimal to binary
5.Conversion decimal to hexadecimal
6.Conversion hexadecimal to decimal
7.Conversion decimal to binary
8.Conversion de binary to hexadecimal

posee una memoria la cual se puede ver con el comando SM cuando este en una funcion mateática.
Puede usar los datos guardados en la memoria usando M1 M2 M3 M4 M5 M6 M7 M8 M9 M10.

Tiene un modo de flujo de operaciones. en la funcion 18, y para salir se digita #.
